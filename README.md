# A Comprehensive Measure of Well-Being
## Human Development Index (HDI) Predictor

A machine learning web application that predicts a country's Human Development Index using life expectancy, schooling years, and GNI per capita based on the official UNDP formula.

### Tech Stack
- React + TypeScript
- Vite + TanStack Router
- Tailwind CSS
- UNDP HDI Formula (geometric mean of health, education, and income indices)

### HDI Tiers
- 🟢 Very High: HDI ≥ 0.800
- 🔵 High: HDI ≥ 0.700
- 🟡 Medium: HDI ≥ 0.550
- 🔴 Low: HDI < 0.550
