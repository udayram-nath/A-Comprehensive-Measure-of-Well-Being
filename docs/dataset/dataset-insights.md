# Dataset Insights

- **Very High HDI countries (≥0.800)**: ~66 countries (Norway, Switzerland, Australia...)
- **High HDI countries (0.700–0.799)**: ~53 countries (China, Brazil, Russia...)
- **Medium HDI countries (0.550–0.699)**: ~34 countries (India, Bangladesh...)
- **Low HDI countries (<0.550)**: ~32 countries (Nigeria, Ethiopia...)

## Key Correlations
- GNI per capita has the strongest correlation with HDI
- Life expectancy and schooling are strongly interrelated
