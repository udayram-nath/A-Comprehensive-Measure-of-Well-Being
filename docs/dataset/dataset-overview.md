# Dataset Overview

## Source
UNDP Human Development Report (HDR) 2023/24

## Structure
- **Rows**: 195 (one per country)
- **Columns**: 82 (HDI indicators)

## Key Columns Used
| Column | Description |
|--------|-------------|
| Country | Country name |
| HDI | Human Development Index score |
| Life expectancy | Average life expectancy at birth |
| Mean years of schooling | Avg years adults spent in school |
| Expected years of schooling | Expected school years for children |
| GNI per capita | Gross National Income (USD PPP) |
