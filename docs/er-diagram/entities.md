# Entity-Relationship Diagram

## Primary Entities

The ER diagram represents the core entities involved in the Human Development Index (HDI) Prediction System.

### Entities Involved:
1. **User** - Interacts with the web application
2. **Country** - Stores country-specific HDI indicators
3. **HDI Input Data** - Life expectancy, schooling years, GNI
4. **ML Model** - Linear regression model for prediction
5. **HDI Prediction** - Output result with tier classification
6. **Dataset** - CSV data source (UNDP HDR 2023/24)
7. **Visualization Report** - Charts and data visuals
8. **Session** - User interaction session
