# Technical Architecture

## Layers

### Frontend Layer
- React 19 + TypeScript
- TanStack Router (file-based routing)
- Tailwind CSS for styling

### Application Layer
- UNDP HDI formula (geometric mean)
- Client-side computation
- State management via React hooks

### Data Layer
- Country presets from UNDP HDR 2023/24
- CSV dataset (HDI.csv - 195 countries, 82 columns)
