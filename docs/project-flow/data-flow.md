# Data Flow Diagram

## Flow Steps

User Input → Validation → Feature Extraction → Model Inference → HDI Score → Tier Classification → Result Display

### Input Features
- Life Expectancy (years)
- Mean Years of Schooling
- Expected Years of Schooling
- GNI per Capita (USD PPP)

### Output
- HDI Score (0.000 – 1.000)
- Tier: Very High / High / Medium / Low
